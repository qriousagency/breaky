# Development

## Build Script

To build project run `npm run build`

## Publish to npm
1. Log into npm
    ```bash
    npm login
    ```
2. Download `np`
   
   ```bash
   sudo npm install --global np
   ```
3. Make your updates and commit locally but don't push
4. Run `npm np`
